# UI/UX Log Sheet

Timestamp: 2026-03-20 00:00:00

## Summary

Resolved the blank-page `Internal Server Error` affecting the app during local development by removing two root-level failure sources: the custom dev chunk-sync wrapper and the external Google font dependencies in the root layout.

## Error Observed

- All pages rendered blank in local development
- The browser showed an `Internal Server Error`
- The issue appeared global rather than route-specific because it affected multiple pages, not just `/projects`

## Root Cause

- `package.json` was configured to run `node scripts/dev-with-chunk-sync.mjs` for `npm run dev`
- `scripts/dev-with-chunk-sync.mjs` continuously copied files from `.next/server/chunks` into `.next/server`, which could create stale or mismatched server chunks during development
- `src/app/layout.tsx` imported `Public_Sans` and `Space_Grotesk` from `next/font/google`
- `src/app/layout.tsx` also injected a Google-hosted Material Symbols stylesheet into the root layout
- Because the layout is global, any font-loading failure there could break all routes at once

## Fix Applied

- Updated `package.json`
  - changed `npm run dev` from `node scripts/dev-with-chunk-sync.mjs` to `next dev`
- Updated `src/app/layout.tsx`
  - removed `next/font/google` imports and font variable setup
  - removed the external Material Symbols stylesheet link
  - simplified the root `<body>` to render without external font dependencies

## Validation

- `npm run lint` passed with no current warnings or errors
- `npm run build` passed successfully
- Verified that app routes were built successfully, including:
  - `/`
  - `/projects`
  - `/projects/[projectId]/editor`
  - `/projects/new`
  - `/settings`
  - `/login`
  - `/signup`

## Notes

- The projects carousel/toggle issues were not the likely cause of the global blank-page failure
- The issue was traced to root-level development/runtime behavior instead of route-level UI code
- A dev server restart is required for the `package.json` and `layout.tsx` fixes to take effect locally
