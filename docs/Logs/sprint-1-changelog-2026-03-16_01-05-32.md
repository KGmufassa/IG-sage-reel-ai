# Sprint 1 Change Log

Timestamp: 2026-03-16 01:05:32

## Summary

Implemented Sprint 1 backend foundation for Parallax Story Composer using Next.js, Prisma, and a PostgreSQL-oriented architecture.

## Changes Made

- Scaffolded a new Next.js TypeScript application foundation with core project config in `package.json`, `next.config.ts`, `tsconfig.json`, `next-env.d.ts`, `.gitignore`, and `.env.example`.
- Added a base app shell and status landing page in `src/app/layout.tsx`, `src/app/page.tsx`, and `src/app/globals.css`.
- Implemented typed environment and config handling in `src/config/env.ts` and `src/config/constants.ts`.
- Added centralized backend foundation utilities for structured errors, request context, correlation IDs, route wrapping, logging, and API responses in `src/core/*` and `src/lib/api-response.ts`.
- Added infrastructure setup for Prisma and in-memory rate limiting in `src/infrastructure/db/prisma.ts` and `src/infrastructure/rate-limit/memory-rate-limiter.ts`.
- Implemented Sprint 1 feature modules for authentication, guest sessions, authorization policies, and project bootstrap flows in `src/modules/auth/*`, `src/modules/guest-sessions/*`, `src/modules/authorization/*`, and `src/modules/projects/*`.
- Added HTTP controllers and versioned API routes for liveness, readiness, auth session lookup, registration, guest session issuance, and project creation in `src/interfaces/http/controllers/*` and `src/app/api/v1/*`.
- Added Auth.js integration in `src/auth.ts` with credentials auth enabled by default and optional Google auth when environment variables are configured.
- Added Prisma schema and initial migration for Sprint 1 domain models in `prisma/schema.prisma` and `prisma/migrations/20260316000000_init/migration.sql`.
- Added lint/test tooling and unit tests in `eslint.config.mjs`, `vitest.config.ts`, `tests/unit/modules/auth/validator.test.ts`, and `tests/unit/modules/authorization/policy.test.ts`.

## Verification Completed

- `npm run lint` passed.
- `npm test` passed.
- `npx prisma validate` passed when `DATABASE_URL` was provided.
- `npm run build` passed with required environment variables set.

## Notes

- Existing unrelated repo changes in docs and `.opencode` were left untouched.
- Google auth is conditional and activates only when `GOOGLE_CLIENT_ID` and `GOOGLE_CLIENT_SECRET` are configured.
- This Sprint 1 work establishes the backend base needed for later project, upload, scene, claim, and processing flows.
